# Vote Eligibility Checker

A straightforward web tool to check voting eligibility instantly. This project is built using vanilla HTML, CSS, and JavaScript.

## Features
- **Instant Feedback:** Get results immediately upon clicking the check button.
- **Input Validation:** Handles empty fields and invalid age inputs.
- **Responsive Design:** Works seamlessly on both desktop and mobile devices.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## How to Run
1. Clone the repository.
2. Open `https://muhammadzaid-kh.github.io/Vote-Eligibility-Checker/` in any web browser.
